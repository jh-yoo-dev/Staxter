Stack

  <li>Spring Boot 2.2.6.RELEASE</li>

database setting
-----------------------
<ul>
  <li>server port : 8090
  <li>spring.h2.console.enabled=true</li>
  <li>spring.h2.console.path=/console</li>

-----------------------
<ul>

    <li>mvn install</li>
    <li>mvn package</li>
    <li>java -jar demo-0.0.1-SNAPSHOT.jar</li>
</ul>
